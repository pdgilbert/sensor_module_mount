# sensor_module_mount

##  Status: DRAFT WIP

## Summary
This repository has FreeCad python code for a simple 3D printed support to mount 
sensor modules onto electrical outlet cover plates. The sensor modules and larger
project are described at https://github.com/pdgilbert/SensorProject.

The mount is X shaped frame with mounting points on the end of each leg. This is for 
sensor module with mount holes 43.5mm apart in x direction and 66.5 mm in y direction.

The `mount.stl` file is generated with FreeCad using the `mount.py` code. More instruction
for doing this are included as comments in the code.

The `stl` file can be downloaded directly and used for 3D printing.
